1. Data Collection & Cleaning

2. Exploratory Data Analysis (EDA)

- Visualized stock price trends over time using Plotly.

- Computed essential financial metrics for each company:

- Volatility: Standard deviation of closing prices (risk indicator).

- Growth Rate: Average daily % change in price.

- ROI: Percentage change from first to last available price.

3. Mutual Fund Plan Creation ‚Äî High ROI + Low Risk Strategy

- ROI above median

- Starting monthly investment: ‚Çπ5,000
